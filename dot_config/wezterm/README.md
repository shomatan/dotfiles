# WezTerm設定

tmux風のキーバインドを設定したWezTerm設定です。

## 基本設定

- リーダーキー: `Ctrl+a`（tmux風）
- フォントサイズ: 12.0
- 背景透過: 80%
- IME: 有効

## キーバインド

### リーダーキー（Ctrl+a）を使った操作

#### ペイン分割
| キー | 動作 |
|------|------|
| `Leader -` | 縦分割 |
| `Leader \` | 横分割 |
| `Leader \|` | 横分割（Shift） |

#### ペイン移動
| キー | 動作 |
|------|------|
| `Leader h/j/k/l` | 左/下/上/右のペインへ |
| `Leader o` | 次のペインへ |

#### ペインサイズ調整
| キー | 動作 |
|------|------|
| `Leader H/J/K/L` | 各方向に拡大 |

#### ペイン操作
| キー | 動作 |
|------|------|
| `Leader z` | ペインズーム切り替え |
| `Leader x` | ペインを閉じる |
| `Leader q` | ペイン選択モード |
| `Leader Space` | ペインを回転 |

#### タブ操作
| キー | 動作 |
|------|------|
| `Leader c` | 新しいタブ |
| `Leader n` | 次のタブ |
| `Leader p` | 前のタブ |
| `Leader 0-9` | タブ番号で移動 |
| `Leader &` | タブを閉じる |

#### コピー・ペースト
| キー | 動作 |
|------|------|
| `Leader [` | コピーモード |
| `Leader ]` | ペースト |

#### その他
| キー | 動作 |
|------|------|
| `Leader r` | 設定再読み込み |
| `Leader :` | コマンドパレット |
| `Leader Ctrl+a` | Ctrl+aを送信 |

### Macショートカット（Cmd）

| キー | 動作 |
|------|------|
| `Cmd+c` | コピー |
| `Cmd+v` | ペースト |
| `Cmd+t` | 新しいタブ |
| `Cmd+w` | 閉じる |
| `Cmd+n` | 新しいウィンドウ |
| `Cmd+1-9` | タブ切り替え |
| `Cmd+f` | 検索 |
| `Cmd+Enter` | フルスクリーン |
| `Cmd+=/-/0` | フォント拡大/縮小/リセット |

### Ctrl操作（Neovimと統一）

| キー | 動作 |
|------|------|
| `Ctrl+h/j/k/l` | ペイン移動 |
| `Ctrl+n/p` | 次/前のタブ |

### Ctrl+Shift操作

| キー | 動作 |
|------|------|
| `Ctrl+Shift -` | 縦分割 |
| `Ctrl+Shift \|` | 横分割 |
| `Ctrl+Shift z` | ペインズーム |

### Git Worktree操作

| キー | 動作 |
|------|------|
| `Ctrl+Shift w` | Worktree選択 |
| `Ctrl+Shift c` | Worktree作成 |
| `Ctrl+Shift d` | Worktree削除 |

## コピーモード（Vim風）

| キー | 動作 |
|------|------|
| `h/j/k/l` | カーソル移動 |
| `w/b/e` | 単語移動 |
| `0/$` | 行頭/行末 |
| `g/G` | バッファ先頭/末尾 |
| `v/V/Ctrl+v` | 文字/行/矩形選択 |
| `y` | ヤンク（コピー） |
| `/` | 検索 |
| `q/Escape` | 終了 |
